Resumen Ejecutivo

Un proyecto diseñado para atraer clientes, no solo existir online

Este documento presenta la propuesta completa para el desarrollo del sitio web de Patricia de Ávila, psicóloga. El objetivo central no es solo crear una web bonita, sino construir una herramienta de captación activa: un sitio que aparezca en Google y Bing cuando alguien busca psicólogo en su área, que inspire confianza desde el primer segundo y que convierta visitas en consultas reales.

El proyecto incluye un sitio web de página única, completamente personalizado, optimizado para todos los dispositivos, con un plan de posicionamiento en buscadores desde el primer día y la creación de un perfil en Google Maps que refuerza la presencia local.

Entregables

Qué se incluye en el proyecto

01

Sitio Web One-Page Personalizado

Diseño único, creado a medida para Patricia de Ávila. Nada de plantillas genéricas. El sitio refleja su identidad profesional y habla directamente a sus pacientes potenciales.

HTML · CSS · JavaScript Diseño a medida Animaciones sutiles Totalmente responsive
02

Alojamiento Gratuito (Sin coste de hosting)

El sitio se publicará en GitHub Pages o Netlify, plataformas profesionales de alojamiento estático 100% gratuitas, con HTTPS incluido y alta disponibilidad. No hay cuota mensual de hosting.

GitHub Pages Netlify (alternativa) HTTPS incluido Sin cuota mensual
03

Dominio Propio (Recomendado)

Se recomienda contratar un dominio propio como patriciadeavila.es para mayor profesionalismo y SEO. Coste aproximado: 10–15 €/año. La configuración con el hosting gratuito está incluida en el proyecto.

.es o .com ~12€/año Configuración incluida
04

SEO On-Page Completo

Optimización técnica para Google y Bing desde el primer día: metaetiquetas, Open Graph, schema.org para profesionales de la salud, velocidad de carga optimizada y sitemap XML para indexación.

Meta SEO Schema.org Sitemap XML Core Web Vitals Open Graph
05

Alta en Google Search Console & Bing Webmaster

Registro del sitio en las herramientas oficiales de ambos buscadores para garantizar la indexación correcta y el seguimiento del posicionamiento.

Google Search Console Bing Webmaster Tools Indexación inmediata
06

Perfil de Google Business (Google Maps)

Creación y optimización del perfil de negocio local en Google Maps: categoría, descripción, horarios, fotos, dirección y número de contacto. Fundamental para aparecer en búsquedas locales del tipo "psicóloga en [ciudad]".

Google Business Profile Google Maps SEO Local Reseñas
Arquitectura del Sitio

Las secciones del sitio web

La página única está estructurada para guiar al visitante de forma natural hacia el contacto.

✨

Hero / Portada

Primera impresión impactante con titular claro, propuesta de valor y llamada a la acción principal (CTA).

🌿

Sobre Patricia

Biografía profesional, formación, valores y enfoque terapéutico. Humaniza a la psicóloga y genera confianza.

💼

Servicios

Áreas de especialización: ansiedad, depresión, terapia de pareja, etc. Descripción clara de cada servicio.

⭐

Testimonios / Reseñas

Opiniones de pacientes (anonimizadas si se prefiere). El elemento de mayor conversión para nuevos clientes.

📬

Contacto

Formulario de contacto, teléfono, email, mapa integrado y enlace directo a WhatsApp si se desea.

🔗

Footer

Aviso legal, política de privacidad (obligatorio RGPD en España), redes sociales y copyright.
